librdf

Resource Description Framework (RDF) application framework

Dave Beckett

OVERVIEW

librdf is an application framework for RDF that allows plugging in of various modules to support different parsers, storage mechanisms or models.

This library is alpha quality. See the current list of bugs or the up to date copy, the TODO list and the FAQS for more information.
