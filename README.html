Redland - An RDF Application Framework

Dave Beckett

OVERVIEW

librdf is an application framework for RDF that allows plugging in of various modules to support different parsers, storage mechanisms or models.

This library is alpha quality. See the todo list or the website version for an up-to-date copy, and the FAQS for more information.
