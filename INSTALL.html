librdf - Installation instructions

Getting the sources

1. From CVS

(if you have access to it - ILRT private at present) 

    export CVSROOT=:pserver:LOGIN@cvs.desire.org:/usr/local/cvsroot
    export CVS_RSH=ssh
    cvs login
    ...
    cvs checkout librdf
where LOGIN is your login on the CVS server.

You will probably then need to create the automake and autoconf derived files, as described below.

Configuring the package

If there is no configure program, you can create it with: 

  automake --add-missing; aclocal autoheader; autoconf

Building

See also the generic GNU installation instructions in INSTALL 

   ./configure
   make

Testing

Build and run the built in tests with: 

    make check
